Waters (.raw)
=============

Waters directories are distinguished by the .raw extension. 

**rainbow** can parse the following Waters binary formats:

.. toctree::
   :maxdepth: 1

   waters/chrodat
   waters/funcdat2
   waters/funcdat6
   waters/funcdat8

It may be useful to search for a binary format by detector. 

.. list-table::
   :header-rows: 1

   * - Detector
     - Formats
   * - UV
     - a
